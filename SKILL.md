# IDcard-smjj - 私募基金管理人销售卡片生成器

**ID:** IDcard-smjj  
**版本:** 1.0.0  
**作者:** 赖克宝 🐸  

## 功能概述

根据尽调报告自动生成符合标准格式的私募基金管理人销售卡片（.docx），包含12标准板块 + 万神殿分析（4部分）。

## 触发条件

当用户需要：
- 制作私募销售卡片
- 生成管理人资料卡
- 创建机构销售工具
- 制作尽调报告摘要卡片

## 标准输出

12板块结构：
1. 基本信息
2. 产品线详解
3. 基金经理实战复盘
4. 核心优势
5. 风险提示
6. 市场环境与配置建议
7. **【万神殿分析】**（4部分完整）
8. 投资建议
9. 客户常见问题（FAQ）
10. 舆情小插曲
11. 投资者真实评价
12. 附录

## 依赖

- **AlphaGBM/investment-masters** - 万神殿分析的10位大师数据
- **docx** npm包 - Word文档生成
- **pdfplumber** - PDF内容提取（如需要）

## 使用方法

### 方式1：直接调用（推荐）

```javascript
const { generateSalesCard } = require('./index.js');

const data = {
  // 按12板块结构填写数据
  basic: { /* 基本信息 */ },
  products: [ /* 产品线 */ ],
  cases: [ /* 实战复盘 */ ],
  advantages: [ /* 核心优势 */ ],
  risks: [ /* 风险提示 */ ],
  marketFit: [ /* 市场匹配度 */ ],
  recommendation: { /* 投资建议 */ },
  faq: [ /* FAQ */ ],
  news: [ /* 舆情 */ ],
  reviews: [ /* 投资者评价 */ },
  appendix: { /* 附录 */ },
  pantheon: { /* 万神殿分析 - 自动生成 */ }
};

await generateSalesCard(data, '输出路径/机构名_销售卡片.docx');
```

### 方式2：CLI命令行

```bash
# 从尽调报告PDF生成
node skill.js --input "尽调报告.pdf" --output "销售卡片.docx"

# 从JSON数据生成
node skill.js --data "data.json" --output "销售卡片.docx"
```

## 数据收集指南

### 信息来源优先级

```
第一优先级：尽调报告
├─ 基本信息 → 板块一
├─ 产品线 → 板块二
├─ 基金经理背景 → 板块三
├─ 投资案例 → 板块三
├─ 优势风险 → 板块四、五
└─ QA问答 → 板块八

第二优先级：网络搜索验证
• "机构名 + 核心人物 + 背景"
• "机构名 + 股权结构"
• "机构名 + 业绩"
• "机构名 + 备案编号"

第三优先级：专业数据库
• 私募排排网（dc.simuwang.com）
• 企查查/天眼查
• 公司官网
```

## 视觉设计规范

### 配色
- 主色：`#1E3A5F`（深蓝）- 标题
- 辅色：`#2E5984`（中蓝）- 次级标题
- 强调色：`#D4AF37`（金色）- 万神殿分析
- 背景色：`#F5F5F5`（浅灰）- 表格交替行
- 成功色：`#27AE60`（绿色）- 高匹配
- 警告色：`#F39C12`（橙色）- 中匹配
- 危险色：`#E74C3C`（红色）- 低匹配

### 图标
- ★ 核心优势
- ⚠ 风险提示
- 📰 舆情新闻
- ◆ 独特性
- ▶ 案例
