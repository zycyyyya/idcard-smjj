# IDcard-smjj 🐸

私募基金管理人销售卡片生成器 - 自动生成符合标准格式的销售工具卡（.docx）

## 功能特点

- ✅ **12标准板块**：完整覆盖销售卡片所需的所有内容板块
- ✅ **万神殿分析**：集成AlphaGBM 10位投资大师框架对照
- ✅ **视觉设计规范**：统一配色、排版、图标系统
- ✅ **无中间文件**：直接输出干净的.docx文件
- ✅ **GitHub Ready**：标准skill结构，可直接上传

## 安装

```bash
# 克隆仓库
git clone https://github.com/yourusername/idcard-smjj.git
cd idcard-smjj

# 安装依赖
npm install
```

## 使用方法

### 方式1：作为Node.js模块使用

```javascript
const { generateSalesCard } = require('./index.js');

const data = {
  basic: {
    name: "滴海基金",
    fullName: "上海滴海私募基金管理有限公司",
    // ... 其他字段
  },
  products: [...],
  cases: [...],
  // ... 其他板块
  pantheon: {
    mastersComparison: [...],
    principles: [...],
    uniqueness: [...],
    quotes: [...]
  }
};

await generateSalesCard(data, '滴海基金_销售卡片.docx');
```

### 方式2：CLI命令行

```bash
# 从JSON数据生成
node skill.js --data 机构数据.json --output 销售卡片.docx

# 查看帮助
node skill.js --help
```

## 12板块结构

| 序号 | 板块 | 关键要求 |
|------|------|----------|
| 1 | 基本信息 | 必填：全称、成立时间、备案编号、规模、资本、核心团队 |
| 2 | 产品线详解 | 4列表格：策略/风格/业绩/备注 |
| 3 | 基金经理实战复盘 | 3个案例：日期+事件+结果 |
| 4 | 核心优势 | 3条，每条有数据支撑 |
| 5 | 风险提示 | 4条，具体不笼统 |
| 6 | 市场环境与配置建议 | 市场主线+匹配度表格+配置建议 |
| 7 | **【万神殿分析】** | **4部分完整** |
| 8 | 投资建议 | 定位/客户/期限/产品/搭配 |
| 9 | 客户常见问题 | 4个尖锐问题+层次回答 |
| 10 | 舆情小插曲 | 3-4条新闻/趣闻 |
| 11 | 投资者真实评价 | 3条评价+来源 |
| 12 | 附录 | 股权/履历/师承 |
